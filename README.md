# crud-table-vue3

### Установка зависимостей
```
yarn install
```
### Запуск клиента
```
yarn dev
```
### Запуск сервера json-sever
```
yarn server
```
### Требует доработки
1. Сортировка работает только в одну сторону 
2. При применении сортировки перестает работать пагинация
